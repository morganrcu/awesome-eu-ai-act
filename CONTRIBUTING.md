# Contributing to Awesome EU AI Act

Thank you for helping keep this list accurate and useful.

## How to Add an Entry

1. Fork this repository
2. Edit `README.md` — add your entry in the appropriate section, alphabetically
3. Format: `**[Name](URL)** — One-sentence description. Key differentiator.`
4. Open a Pull Request with the title: `Add [Tool Name] to [Section Name]`

## Criteria for Inclusion

- **Actively maintained** — updated within the last 12 months
- **Directly relevant** — EU AI Act compliance, AI governance, or closely related standards (ISO 42001, NIST AI RMF)
- **Open-source:** public repository with OSI-approved license
- **Commercial:** free tier, trial, or fully public documentation

## Criteria for Removal

- No activity for > 12 months
- Project deprecated or abandoned
- No longer relevant to EU AI Act

## Style Guide

- Use sentence case for descriptions
- Be concise — one sentence maximum per description
- Include funding info only if it provides useful context (e.g., major certification, EU AI Pact signatory)
- Do not include pricing information — it changes

## What NOT to Include

- Paid-only tools with no public evaluation path
- Generic ML tools with no AI governance relevance
- Tools not relevant to EU market / EU AI Act specifically
- Astroturfing or self-promotion without genuine community value
